# Getting Started

This guide outlines the basic requirements needed before running the Azure Administrator labs.

---

## Prerequisites

Before beginning the labs, ensure the following:

- An active Azure tenant
- An Azure subscription
- Contributor permissions on the subscription
- Access to the Azure Portal

---

## Azure Portal Access

To access the Azure Portal:

1. Navigate to https://portal.azure.com
2. Sign in with your Azure account.
3. Confirm you can access your subscription.

---

## Recommended Preparation

Before starting the labs, it is helpful to understand the following Azure concepts:

- Resource Groups
- Virtual Networks
- Virtual Machines
- Network Security Groups
- Cost Management

These concepts will be used throughout the lab series.

---

## Lab Structure

Each lab in this repository includes:

- Objective
- Resources Created
- Prerequisites
- Step-by-step build instructions
- Validation steps
- Screenshots
- Cleanup instructions

---

## Cost Awareness

Some labs deploy Azure resources that may incur small costs.

To avoid unnecessary charges:

- Use small VM sizes for testing
- Delete resources after completing labs
- Monitor spending using the **Azure Budget + Cost Alerts** lab
